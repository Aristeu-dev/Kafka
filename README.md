 Kafka with microservice
</h1>

## About
- This is a simple microservice API that receives messages simulating a creation user.

## Technologies
- KafkaJS
- NodeJS

## How to use
`Run:`
- In root directory:
```
    docker compose up
```
- In api directory and user directory:
```
    yarn install
```

`To run:`
- In api directory and user directory: 
```
    yarn dev
```
